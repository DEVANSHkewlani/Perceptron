Philosophy & Goals
Why we need a real target system instead of synthetic injection

The Chaos Lab is the validation environment for the entire DCA. It answers one question: when real infrastructure breaks, does the cognitive system detect, reason about, plan, execute, verify, and learn — fully autonomously — within acceptable time bounds?

💡
Why a real e-commerce stack, not synthetic events?
Synthetic events test if your schema is correct. Real infrastructure failures test if your entire perception-to-remediation pipeline produces the right chain of events: database slowness that cascades into API latency, API latency that causes user rage clicks, rage clicks that appear in browser events. Only a real app produces that causal chain.

Three Laws of the Chaos Lab

⚡ Law 1: Real Infrastructure
Every failure is injected at the actual infrastructure layer — not fabricated in code. Slow a real PostgreSQL query, drop real Docker network packets, exhaust a real Redis memory limit.
🔁 Law 2: Closed Loop
Every injected anomaly must travel the complete path: perception → memory → reasoning → planning → execution → verification → playbook update. The loop closes only when the system self-heals.
📊 Law 3: Measurable SLOs
Every scenario has a detection SLO (how fast the event appears on cognitive.events) and a remediation SLO (how fast the system self-heals). These numbers define whether Phase 13 passes.

ShopCore — The Target E-Commerce System
A realistic microservices platform that generates authentic failure signals

ShopCore is a minimal but structurally complete e-commerce backend. It uses all the same infrastructure as your real production stack would — PostgreSQL, Kafka/Redpanda, Redis, Nginx — so every chaos injection hits real infrastructure and produces real signals.

port 8000 API Gateway
Routes /products, /orders, /users. Emits structured JSON logs. Exposes Prometheus metrics. Auth middleware validates JWTs.
port 8001 Product Service
CRUD on PostgreSQL. Uses asyncpg connection pool (max 10). Prometheus counter on slow queries. Exposes /health.
port 8002 Order Service
Writes orders to PostgreSQL, publishes to Redpanda topic order-events. Has a configurable processing delay injected by chaos.
port 8003 User Service
JWT generation/validation backed by Redis for session store. Brute-force detector publishes to cognitive.events directly.
port 3000 React Frontend
SPA with product listing, cart, checkout. cognitive_sdk.js embedded. Sends Web Vitals and rage clicks to /perception/browser-events.
infra Shared Infrastructure
PostgreSQL (shared DB), Redis (sessions + cache), Redpanda (events), Nginx (reverse proxy + TLS termination), Prometheus + Alertmanager.

"""
ShopCore API Gateway
====================
Routes all external traffic to internal microservices.
Emits structured JSON logs consumed by Fluent Bit → LogAdapter → DCA.

CHAOS HOOKS:
  - CHAOS_LATENCY_MS env var: adds artificial latency to all responses
  - CHAOS_ERROR_RATE env var: percentage of requests to return 500
  - CHAOS_CIRCUIT_OPEN env var: block all upstream calls
"""

from __future__ import annotations

import asyncio, json, logging, os, random, time, uuid
from contextlib import asynccontextmanager
from datetime import datetime, timezone

import httpx
from fastapi import FastAPI, Request, HTTPException, status
from fastapi.middleware.cors import CORSMiddleware
from fastapi.responses import JSONResponse
from prometheus_client import Counter, Histogram, generate_latest, CONTENT_TYPE_LATEST
from starlette.responses import Response

# ── Structured logger → Fluent Bit picks this up ─────────────────────────────
logging.basicConfig(
    format=json.dumps({
        "timestamp": "%(asctime)s",
        "level": "%(levelname)s",
        "service": "api-gateway",
        "message": "%(message)s",
    }),
    level=logging.INFO,
)
logger = logging.getLogger("api-gateway")

# ── Prometheus metrics ────────────────────────────────────────────────────────
REQUEST_COUNT = Counter(
    'shopcore_gateway_requests_total',
    'Total HTTP requests',
    ['method', 'path', 'status']
)
REQUEST_LATENCY = Histogram(
    'shopcore_gateway_latency_seconds',
    'Request latency in seconds',
    ['path'],
    buckets=[.005, .01, .025, .05, .1, .25, .5, 1, 2.5, 5, 10]
)

# ── Upstream service URLs ────────────────────────────────────────────────────
SERVICES = {
    "products": os.getenv("PRODUCT_SERVICE_URL", "http://product-service:8001"),
    "orders":   os.getenv("ORDER_SERVICE_URL",   "http://order-service:8002"),
    "users":    os.getenv("USER_SERVICE_URL",    "http://user-service:8003"),
}

# ── Chaos injection state (read from env at request time) ─────────────────────
def _chaos_latency_ms() -> float:
    return float(os.getenv("CHAOS_LATENCY_MS", "0"))

def _chaos_error_rate() -> float:
    return float(os.getenv("CHAOS_ERROR_RATE", "0"))

def _circuit_open() -> bool:
    return os.getenv("CHAOS_CIRCUIT_OPEN", "false").lower() == "true"


async def _proxy(
    request: Request,
    upstream_base: str,
    path: str,
    client: httpx.AsyncClient,
) -> JSONResponse:
    """Proxy a request to an upstream service with chaos injection."""
    trace_id = str(uuid.uuid4())[:8]

    # Chaos: artificial latency
    latency_ms = _chaos_latency_ms()
    if latency_ms > 0:
        await asyncio.sleep(latency_ms / 1000)
        logger.warning(f"Artificial latency applied: {latency_ms}ms trace_id={trace_id}")

    # Chaos: artificial error rate
    if random.random() < _chaos_error_rate():
        logger.error(f"Chaos error injected for {path} trace_id={trace_id}")
        raise HTTPException(status_code=500, detail="Internal server error (chaos)")

    # Chaos: circuit breaker open
    if _circuit_open():
        logger.error(f"Circuit breaker open — upstream call blocked trace_id={trace_id}")
        raise HTTPException(status_code=503, detail="Service temporarily unavailable")

    start = time.monotonic()
    try:
        body = await request.body()
        resp = await client.request(
            method=request.method,
            url=f"{upstream_base}{path}",
            headers={k: v for k, v in request.headers.items()
                     if k.lower() != "host"},
            content=body,
            params=dict(request.query_params),
        )
        elapsed_ms = (time.monotonic() - start) * 1000
        logger.info(f"Proxied {request.method} {path} → {resp.status_code} in {elapsed_ms:.1f}ms trace_id={trace_id}")
        return JSONResponse(
            content=resp.json() if resp.headers.get("content-type", "").startswith("application/json") else {"body": resp.text},
            status_code=resp.status_code,
        )
    except httpx.ConnectError:
        elapsed_ms = (time.monotonic() - start) * 1000
        logger.error(f"Upstream connection refused for {path} elapsed_ms={elapsed_ms:.0f} trace_id={trace_id}")
        raise HTTPException(status_code=503, detail="Upstream service unreachable")
    except httpx.TimeoutException:
        elapsed_ms = (time.monotonic() - start) * 1000
        logger.error(f"Upstream timeout for {path} elapsed_ms={elapsed_ms:.0f} trace_id={trace_id}")
        raise HTTPException(status_code=504, detail="Upstream request timed out")


async def lifespan(app: FastAPI):
    app.state.client = httpx.AsyncClient(timeout=10.0)
    yield
    await app.state.client.aclose()

app = FastAPI(title="ShopCore API Gateway", lifespan=lifespan)
app.add_middleware(CORSMiddleware, allow_origins=["*"], allow_methods=["*"], allow_headers=["*"])


async def _with_metrics(path: str, handler):
    start = time.monotonic()
    try:
        result = await handler()
        REQUEST_COUNT.labels(method="GET", path=path, status="2xx").inc()
        return result
    except HTTPException as e:
        REQUEST_COUNT.labels(method="GET", path=path, status=str(e.status_code)).inc()
        raise
    finally:
        REQUEST_LATENCY.labels(path=path).observe(time.monotonic() - start)

async def products_proxy(request: Request, path: str = ""):

@app.get("/health")

"""
Product Service
===============
FastAPI + asyncpg. Runs real SQL queries on PostgreSQL.
CHAOS HOOKS (all via environment variables set by chaos engine):
  CHAOS_SLOW_QUERY_MS  — adds pg_sleep() to every SELECT
  CHAOS_POOL_EXHAUST   — holds connections open to exhaust pool
  CHAOS_DB_ERROR_RATE  — random SQL failures
"""

import asyncio, json, logging, os, random, time
from contextlib import asynccontextmanager

import asyncpg
from fastapi import FastAPI, HTTPException
from pydantic import BaseModel
from prometheus_client import Counter, Histogram, Gauge, generate_latest, CONTENT_TYPE_LATEST
from starlette.responses import Response

logger = logging.getLogger("product-service")
logging.basicConfig(
    format=json.dumps({"timestamp": "%(asctime)s", "level": "%(levelname)s", "service": "product-service", "message": "%(message)s"}),
    level=logging.INFO,
)

DB_DSN  = os.getenv("DATABASE_URL", "postgresql://shopcore:shopcore@postgres:5432/shopcore")
POOL_MIN = 2
POOL_MAX = 10

QUERY_LATENCY = Histogram('product_db_query_seconds', 'DB query latency', ['operation'])
DB_POOL_SIZE  = Gauge('product_db_pool_used', 'Used DB connections')
ERROR_COUNTER = Counter('product_errors_total', 'Total errors', ['type'])


class Product(BaseModel):
    name: str
    price: float
    stock: int
    category: str


@asynccontextmanager
async def lifespan(app: FastAPI):
    app.state.pool = await asyncpg.create_pool(
        DB_DSN, min_size=POOL_MIN, max_size=POOL_MAX
    )
    # Ensure schema exists
    async with app.state.pool.acquire() as conn:
        await conn.execute("""
            CREATE TABLE IF NOT EXISTS products (
                id SERIAL PRIMARY KEY,
                name TEXT NOT NULL,
                price NUMERIC(10,2) NOT NULL,
                stock INT NOT NULL DEFAULT 0,
                category TEXT,
                created_at TIMESTAMPTZ DEFAULT NOW()
            )
        """)
        # Seed some data if empty
        count = await conn.fetchval("SELECT COUNT(*) FROM products")
        if count == 0:
            await conn.executemany(
                "INSERT INTO products(name, price, stock, category) VALUES($1,$2,$3,$4)",
                [("Laptop Pro 15", 1299.99, 50, "electronics"),
                 ("Wireless Mouse", 29.99, 200, "peripherals"),
                 ("USB-C Hub", 49.99, 150, "peripherals"),
                 ("4K Monitor", 599.99, 30, "electronics"),
                 ("Mechanical Keyboard", 149.99, 75, "peripherals")],
            )
    yield
    await app.state.pool.close()

app = FastAPI(title="ShopCore Products", lifespan=lifespan)


async def get_conn(pool) -> asyncpg.Connection:
    """Acquire a connection with chaos hooks applied."""
    DB_POOL_SIZE.set(pool.get_size() - pool.get_idle_size())

    # Chaos: pool exhaustion (hold connection open for a long time)
    if os.getenv("CHAOS_POOL_EXHAUST") == "true":
        logger.error(
            "connection pool exhausted — too many connections"
            " database=shopcore service=product-service"
        )
        ERROR_COUNTER.labels(type="pool_exhausted").inc()
        raise HTTPException(status_code=503, detail="DB pool exhausted")

    return await pool.acquire()


@app.get("/products")
async def list_products(request: Request):
    start = time.monotonic()
    conn = await get_conn(request.app.state.pool)
    try:
        slow_ms = float(os.getenv("CHAOS_SLOW_QUERY_MS", "0"))
        if slow_ms > 0:
            logger.warning(f"query timed out exceeded {slow_ms}ms database=shopcore slow_query=true service=product-service")
            await conn.execute(f"SELECT pg_sleep({slow_ms / 1000})")

        rows = await conn.fetch("SELECT * FROM products ORDER BY id")
        elapsed = time.monotonic() - start
        QUERY_LATENCY.labels(operation="list").observe(elapsed)
        return [dict(r) for r in rows]
    finally:
        await request.app.state.pool.release(conn)


@app.get("/products/{product_id}")
async def get_product(product_id: int, request: Request):
    conn = await get_conn(request.app.state.pool)
    try:
        row = await conn.fetchrow("SELECT * FROM products WHERE id=$1", product_id)
        if not row:
            raise HTTPException(status_code=404, detail="Product not found")
        return dict(row)
    finally:
        await request.app.state.pool.release(conn)

async def health(): return {"status": "ok", "service": "product-service"}

@app.get("/metrics")
async def metrics(): return Response(generate_latest(), media_type=CONTENT_TYPE_LATEST)

"""
Order Service
=============
Processes orders: writes to PostgreSQL, publishes to Kafka.
CHAOS HOOKS:
  CHAOS_ORDER_DELAY_MS — artificial delay before processing
  CHAOS_KAFKA_BLOCK    — stop publishing to Kafka (creates lag)
  CHAOS_ORDER_FAIL_RATE — random failure rate
"""

import asyncio, json, logging, os, random, time, uuid
from contextlib import asynccontextmanager

import asyncpg
from aiokafka import AIOKafkaProducer
from fastapi import FastAPI, HTTPException, Request
from pydantic import BaseModel

DB_DSN   = os.getenv("DATABASE_URL", "postgresql://shopcore:shopcore@postgres:5432/shopcore")
KAFKA_BS = os.getenv("KAFKA_BOOTSTRAP", "redpanda:9092")

logger = logging.getLogger("order-service")
logging.basicConfig(
    format=json.dumps({"timestamp": "%(asctime)s", "level": "%(levelname)s", "service": "order-service", "message": "%(message)s"}),
    level=logging.INFO,
)


class OrderRequest(BaseModel):
    user_id: str
    product_ids: list[int]
    total_amount: float


@asynccontextmanager
async def lifespan(app: FastAPI):
    app.state.pool = await asyncpg.create_pool(DB_DSN, min_size=2, max_size=10)
    app.state.producer = AIOKafkaProducer(bootstrap_servers=KAFKA_BS)
    await app.state.producer.start()
    async with app.state.pool.acquire() as conn:
        await conn.execute("""
            CREATE TABLE IF NOT EXISTS orders (
                id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
                user_id TEXT NOT NULL,
                product_ids INTEGER[],
                total_amount NUMERIC(10,2),
                status TEXT DEFAULT 'pending',
                created_at TIMESTAMPTZ DEFAULT NOW()
            )
        """)
    yield
    await app.state.producer.stop()
    await app.state.pool.close()

app = FastAPI(title="ShopCore Orders", lifespan=lifespan)


@app.post("/orders", status_code=201)
async def create_order(order: OrderRequest, request: Request):
    order_id = str(uuid.uuid4())

    # Chaos: artificial processing delay
    delay_ms = float(os.getenv("CHAOS_ORDER_DELAY_MS", "0"))
    if delay_ms > 0:
        logger.warning(f"order processing delayed — slow_query=true elapsed_ms={delay_ms} service=order-service")
        await asyncio.sleep(delay_ms / 1000)

    # Chaos: random failure
    fail_rate = float(os.getenv("CHAOS_ORDER_FAIL_RATE", "0"))
    if random.random() < fail_rate:
        logger.error(f"internal server error — unhandled exception in order processing order_id={order_id}")
        raise HTTPException(status_code=500, detail="Order processing failed")

    async with request.app.state.pool.acquire() as conn:
        await conn.execute(
            "INSERT INTO orders(id, user_id, product_ids, total_amount) VALUES($1,$2,$3,$4)",
            order_id, order.user_id, order.product_ids, order.total_amount,
        )

    # Chaos: block Kafka publish (creates consumer lag)
    if os.getenv("CHAOS_KAFKA_BLOCK") != "true":
        event = {"order_id": order_id, "user_id": order.user_id, "total": order.total_amount, "ts": time.time()}
        await request.app.state.producer.send_and_wait(
            "order-events", json.dumps(event).encode()
        )
    else:
        logger.warning(f"downstream kafka blocked — dependency call failed service=order-service order_id={order_id}")

    return {"order_id": order_id, "status": "created"}

async def health(): return {"status": "ok"}

ChaosController — chaos_engine/controller.py
Central orchestrator for all failure injection. Modifies Docker environment variables and network rules.

The ChaosController injects failures by two mechanisms: (1) setting environment variables on running containers (which the services read at request time via their chaos hooks), and (2) applying Docker network rules for real packet-level failures.

chaos_engine/controller.py
"""
ChaosController
===============
Injects infrastructure failures into ShopCore by manipulating:
  1. Container environment variables (services have chaos hooks that read these)
  2. Docker network rules (tc/iptables for real packet-level failures)
  3. PostgreSQL direct (inject slow queries, lock tables)
  4. Kafka producer (stop publishing to simulate lag)

Each scenario is:
  - activate(params) : inject the failure
  - deactivate()     : restore normal state
  - verify()         : confirm the failure is actually present
"""

class DockerChaos:
    """Helpers for Docker-based failure injection."""

    def __init__(self):
        self.client = docker.from_env()

    def _get_container(self, service_name: str) -> docker.models.containers.Container:
        containers = self.client.containers.list(
            filters={"label": f"com.shopcore.service={service_name}"}
        )
        if not containers:
            # Fallback: name-based lookup
            containers = self.client.containers.list(
                filters={"name": service_name}
            )
        if not containers:
            raise RuntimeError(f"Container for {service_name} not found")
        return containers[0]

    def set_env(self, service_name: str, env_vars: dict[str, str]):
        """Update environment variables on a running container by recreating it."""
        container = self._get_container(service_name)
        image = container.image
        config = container.attrs["Config"]
        current_env = dict(e.split("=", 1) for e in config.get("Env", []))
        current_env.update(env_vars)
        new_env = [f"{k}={v}" for k, v in current_env.items()]

        network_name = f"{DOCKER_COMPOSE_PROJECT}_shopcore"
        name = container.name

        container.stop(timeout=5)
        container.remove()

        self.client.containers.run(
            image,
            name=name,
            environment=new_env,
            network=network_name,
            detach=True,
            ports=config.get("ExposedPorts", {}),
        )
        logger.info(f"[Chaos] Set env on {service_name}: {env_vars}")

    def stop_container(self, service_name: str):
        container = self._get_container(service_name)
        container.stop(timeout=2)
        logger.info(f"[Chaos] Stopped container: {service_name}")

    def start_container(self, service_name: str):
        container = self._get_container(service_name)
        container.start()
        logger.info(f"[Chaos] Started container: {service_name}")

    def apply_network_latency(self, service_name: str, latency_ms: int):
        """Apply tc netem latency inside the container."""
        container = self._get_container(service_name)
        container.exec_run(
            f"tc qdisc add dev eth0 root netem delay {latency_ms}ms",
            privileged=True
        )
        logger.info(f"[Chaos] Applied {latency_ms}ms network latency to {service_name}")

    def remove_network_latency(self, service_name: str):
        container = self._get_container(service_name)
        container.exec_run("tc qdisc del dev eth0 root netem", privileged=True)

from __future__ import annotations

import asyncio, logging, time
from datetime import datetime, timezone

import docker
from aiokafka import AIOKafkaProducer

from cognitive_perception.schema.event import CognitiveEvent, Severity, SourceType

logger = logging.getLogger("service-handler")

# Map source_id → Docker service name
SOURCE_TO_CONTAINER = {
    "svc:api-gateway":     "api-gateway",
    "svc:product-service": "product-service",
    "svc:order-service":   "order-service",
    "svc:user-service":    "user-service",
}


class ServiceRemediationHandler:
    """
    Remediates service-level failures by restarting Docker containers.
    Called by the DCA PlanRunner when a plan includes a 'restart_service' step.
    """

    def __init__(self, producer: AIOKafkaProducer, kafka_topic: str = "cognitive.events"):
        self.producer = producer
        self.topic = kafka_topic
        self.docker_client = docker.from_env()

    async def execute(self, event: CognitiveEvent, action_params: dict) -> dict:
        """
        Execute service remediation.
        action_params: {"action": "restart" | "health_gate" | "scale_up"}
        """
        source_id = event.source_id
        action = action_params.get("action", "restart")
        container_name = SOURCE_TO_CONTAINER.get(source_id)

        if not container_name:
            logger.warning(f"No container mapping for {source_id}")
            return {"success": False, "reason": "no container mapping"}

        start = time.monotonic()

        if action == "restart":
            result = await self._restart_container(container_name)
        elif action == "health_gate":
            result = await self._health_gate(container_name, source_id)
        else:
            result = {"success": False, "reason": f"Unknown action: {action}"}

        elapsed_s = time.monotonic() - start

        # Publish remediation feedback event back to DCA
        feedback_event = CognitiveEvent(
            timestamp=datetime.now(timezone.utc),
            source_type=SourceType.AGENT_EVENT,
            source_id="agent:service-remediation-handler",
            event_type="action_completed" if result["success"] else "action_failed",
            severity=Severity.INFO if result["success"] else Severity.HIGH,
            payload={
                "original_event_id": event.event_id,
                "original_event_type": event.event_type,
                "action": action,
                "container": container_name,
                "elapsed_s": round(elapsed_s, 2),
                **result,
            },
            entity_refs=[source_id, "agent:service-remediation-handler"],
            confidence=0.99,
            tags=["remediation", "service", action],
        )
        await self.producer.send_and_wait(
            self.topic, feedback_event.model_dump_json().encode()
        )
        return result

    async def _restart_container(self, container_name: str) -> dict:
        try:
            containers = self.docker_client.containers.list(filters={"name": container_name})
            if not containers:
                # Try to start stopped container
                stopped = self.docker_client.containers.list(
                    all=True, filters={"name": container_name}
                )
                if stopped:
                    stopped[0].start()
                    logger.info(f"[Remediation] Started stopped container: {container_name}")
                    await self._wait_healthy(container_name)
                    return {"success": True, "action_taken": "started"}
                return {"success": False, "reason": "container not found"}
            else:
                containers[0].restart(timeout=10)
                logger.info(f"[Remediation] Restarted container: {container_name}")
                await self._wait_healthy(container_name)
                return {"success": True, "action_taken": "restarted"}
        except Exception as e:
            logger.error(f"[Remediation] Restart failed for {container_name}: {e}")
            return {"success": False, "reason": str(e)}

    async def _health_gate(self, container_name: str, source_id: str) -> dict:
        """Wait for health check to pass before returning traffic."""
        for _ in range(30):  # up to 60s
            try:
                import httpx
                port_map = {"api-gateway": 8000, "product-service": 8001, "order-service": 8002, "user-service": 8003}
                port = port_map.get(container_name, 8000)
                async with httpx.AsyncClient(timeout=2) as c:
                    r = await c.get(f"http://localhost:{port}/health")
                    if r.status_code == 200:
                        return {"success": True, "action_taken": "health_gate_passed"}
            except: pass
            await asyncio.sleep(2)
        return {"success": False, "reason": "health gate timeout"}

    async def _wait_healthy(self, container_name: str, timeout_s: int = 30):
        await self._health_gate(container_name, container_name)

"""
Database Remediation Handler
=============================
Handles slow_query_detected, connection_pool_exhausted,
lock_wait_timeout, connection_leak_detected events.
Actions: kill slow query PID, terminate idle connections, reset pool.
"""

from __future__ import annotations
import asyncio, logging, time
from datetime import datetime, timezone

import asyncpg
from aiokafka import AIOKafkaProducer
from cognitive_perception.schema.event import CognitiveEvent, Severity, SourceType

logger = logging.getLogger("database-handler")


class DatabaseRemediationHandler:
    """Handles database-level failures by executing corrective SQL."""

    def __init__(
        self,
        db_dsn: str,
        producer: AIOKafkaProducer,
        kafka_topic: str = "cognitive.events",
    ):
        self.db_dsn = db_dsn
        self.producer = producer
        self.topic = kafka_topic

    async def execute(self, event: CognitiveEvent, action_params: dict) -> dict:
        action = action_params.get("action", "kill_slow_queries")
        start = time.monotonic()

        try:
            conn = await asyncpg.connect(self.db_dsn, timeout=10)
            try:
                if action == "kill_slow_queries":
                    result = await self._kill_slow_queries(conn, event)
                elif action == "terminate_idle_connections":
                    result = await self._terminate_idle_connections(conn)
                elif action == "kill_blocking_query":
                    result = await self._kill_blocking_query(conn, event)
                else:
                    result = {"success": False, "reason": f"Unknown: {action}"}
            finally:
                await conn.close()
        except Exception as e:
            result = {"success": False, "reason": str(e)}

        elapsed_s = time.monotonic() - start
        await self._publish_feedback(event, action, result, elapsed_s)
        return result

    async def _kill_slow_queries(self, conn: asyncpg.Connection, event: CognitiveEvent) -> dict:
        """Terminate queries running longer than 10 seconds."""
        rows = await conn.fetch("""
            SELECT pid, query_start,
                   EXTRACT(EPOCH FROM now() - query_start) AS duration_s,
                   left(query, 100) AS q
            FROM pg_stat_activity
            WHERE state != 'idle'
              AND pid != pg_backend_pid()
              AND EXTRACT(EPOCH FROM now() - query_start) > 10
            ORDER BY duration_s DESC
        """)
        killed = 0
        for row in rows:
            await conn.execute("SELECT pg_terminate_backend($1)", row["pid"])
            killed += 1
            logger.info(f"[DBHandler] Killed slow query PID {row['pid']} ({row['duration_s']:.0f}s)")
        return {"success": True, "killed_queries": killed}

    async def _terminate_idle_connections(self, conn: asyncpg.Connection) -> dict:
        """Terminate idle-in-transaction connections older than 60 seconds."""
        result = await conn.execute("""
            SELECT pg_terminate_backend(pid)
            FROM pg_stat_activity
            WHERE state = 'idle in transaction'
              AND pid != pg_backend_pid()
              AND EXTRACT(EPOCH FROM now() - query_start) > 60
        """)
        count = int(result.split()[1])
        logger.info(f"[DBHandler] Terminated {count} idle-in-transaction connections")
        return {"success": True, "terminated_connections": count}

    async def _kill_blocking_query(self, conn: asyncpg.Connection, event: CognitiveEvent) -> dict:
        """Kill the blocking PID from a lock_wait_timeout event."""
        blocking_pid = event.payload.get("blocking_pid")
        if not blocking_pid:
            return {"success": False, "reason": "no blocking_pid in event payload"}
        await conn.execute("SELECT pg_terminate_backend($1)", blocking_pid)
        logger.info(f"[DBHandler] Killed blocking query PID {blocking_pid}")
        return {"success": True, "killed_blocking_pid": blocking_pid}

    async def _publish_feedback(self, event, action, result, elapsed_s):
        feedback = CognitiveEvent(
            timestamp=datetime.now(timezone.utc),
            source_type=SourceType.AGENT_EVENT,
            source_id="agent:database-remediation-handler",
            event_type="action_completed" if result["success"] else "action_failed",
            severity=Severity.INFO if result["success"] else Severity.HIGH,
            payload={"original_event_id": event.event_id, "action": action, "elapsed_s": round(elapsed_s, 2), **result},
            entity_refs=[event.source_id, "agent:database-remediation-handler"],
            confidence=0.99,
            tags=["remediation", "database", action],
        )
        await self.producer.send_and_wait(self.topic, feedback.model_dump_json().encode())

class ActionRegistry:
    """
    Resolves which handler to call for a given event_type.
    Called by the DCA PlanRunner during plan execution.
    """

    def __init__(
        self,
        producer: AIOKafkaProducer,
        db_dsn: str = "postgresql://shopcore:shopcore@localhost:5432/shopcore",
        redis_url: str = "redis://localhost:6379",
        kafka_bootstrap: str = "localhost:9092",
    ):
        self._producer = producer
        self._db_dsn = db_dsn
        self._redis_url = redis_url
        self._kafka_bootstrap = kafka_bootstrap
        self._handler_instances: dict[type, Any] = {}

    def _get_handler(self, handler_class: type) -> Any:
        if handler_class not in self._handler_instances:
            if handler_class == ServiceRemediationHandler:
                self._handler_instances[handler_class] = ServiceRemediationHandler(self._producer)
            elif handler_class == DatabaseRemediationHandler:
                self._handler_instances[handler_class] = DatabaseRemediationHandler(self._db_dsn, self._producer)
            elif handler_class == CacheRemediationHandler:
                self._handler_instances[handler_class] = CacheRemediationHandler(self._redis_url, self._producer)
            elif handler_class == QueueRemediationHandler:
                self._handler_instances[handler_class] = QueueRemediationHandler(self._kafka_bootstrap, self._producer)
            elif handler_class == ConfigRemediationHandler:
                self._handler_instances[handler_class] = ConfigRemediationHandler(self._producer)
        return self._handler_instances[handler_class]

    def get_handler_for_event(self, event_type: str) -> tuple[Any, dict] | None:
        if event_type not in REMEDIATION_REGISTRY:
            return None
        entry = REMEDIATION_REGISTRY[event_type]
        handler = self._get_handler(entry.handler_class)
        return handler, entry.action_params

    def requires_approval(self, event_type: str) -> bool:
        return REMEDIATION_REGISTRY.get(event_type, RemediationEntry(None, {}, "")).requires_approval

"""Run after the test suite to print a Phase 13 pass/fail scorecard."""
import asyncio, json
from pathlib import Path

def score(results_json: str):
    results = json.loads(Path(results_json).read_text())
    detection_pass = results["detection_pass"]
    detection_total = results["detection_total"]
    remediation_pass = results["remediation_pass"]
    remediation_total = results["remediation_total"]

    pct_d = detection_pass / detection_total * 100
    pct_r = remediation_pass / remediation_total * 100

    print(f"\n{'='*50}")
    print(f"  PHASE 13 CHAOS LAB SCORECARD")
    print(f"{'='*50}")
    print(f"  Detection:    {detection_pass}/{detection_total} ({pct_d:.0f}%)  {'✅ PASS' if pct_d >= 90 else '❌ FAIL'}")
    print(f"  Remediation:  {remediation_pass}/{remediation_total} ({pct_r:.0f}%)  {'✅ PASS' if pct_r >= 75 else '❌ FAIL'}")
    phase_pass = pct_d >= 90 and pct_r >= 75
    print(f"\n  PHASE 13: {'✅ PASSED — Ready for Phase 14' if phase_pass else '❌ FAILED — Fix SLOs above'}")
    print(f"{'='*50}\n")
    return phase_pass

if __name__ == "__main__":
    import sys
    sys.exit(0 if score(sys.argv[1]) else 1)
